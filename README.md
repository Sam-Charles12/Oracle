# Oracle — Predictive Maintenance Dashboard

A Vite + React + Tailwind dashboard for predictive maintenance data visualization and simulation.

## Quickstart

Prerequisites: Node.js (recommended LTS).

Install dependencies:

```bash
npm install
```

Run the dev server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

## Notes

- A root `.gitignore` was added to exclude `node_modules/` from Git.
- If you previously committed `node_modules`, untrack it with:

```bash
git rm -r --cached node_modules
git commit -m "Stop tracking node_modules"
```
